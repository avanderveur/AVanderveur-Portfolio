Job Application Organizer

The Job Application Organizer is a web application designed to assist veterans in managing their job applications efficiently. Users can add, edit, and delete job applications while tracking important details such as job titles, companies, deadlines, and notes. The app features a user-friendly interface with filtering options to help users easily search and categorize their applications based on status (Applied, Interview, Rejected). Utilizing local storage, the application ensures that user data persists across sessions, making it a valuable tool for navigating the job search process. Technologies Used: HTML, CSS, JavaScript, Local Storage
